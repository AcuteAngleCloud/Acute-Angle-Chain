\hypertarget{group__aacclienttool_intro}{}\subsection{Introduction to claac}\label{group__aacclienttool_intro}
{\ttfamily claac} is a command line tool that interfaces with the R\+E\+ST api exposed by nodaac. In order to use {\ttfamily claac} you will need to have a local copy of {\ttfamily nodaac} running and configured to load the \textquotesingle{}\mbox{\hyperlink{classaacio_1_1chain__api__plugin}{aacio\+::chain\+\_\+api\+\_\+plugin}}\textquotesingle{}.

claac contains documentation for all of its commands. For a list of all commands known to claac, simply run it with no arguments\+: 
\begin{DoxyCode}
$ ./claac
Command Line Interface to AACIO Client
Usage: programs/clAcuteAngleChain/claac [OPTIONS] SUBCOMMAND

Options:
  -h,--help                   \mbox{\hyperlink{struct_print}{Print}} \textcolor{keyword}{this} help message and exit
  -\mbox{\hyperlink{struct_h}{H}},--host TEXT=localhost    the host where nodaac is running
  -p,--port UINT=8888         the port where nodaac is running
  --\mbox{\hyperlink{namespacewallet}{wallet}}-host TEXT=localhost
                              the host where kaacd is running
  --\mbox{\hyperlink{namespacewallet}{wallet}}-port UINT=8888     the port where kaacd is running
  -v,--verbose                output verbose actions on \mbox{\hyperlink{structerror}{error}}

Subcommands:
  version                     Retrieve version information
  create                      Create various items, on and off the blockchain
  \textcolor{keyword}{get}                         Retrieve various items and information from the blockchain
  \textcolor{keyword}{set}                         Set or update blockchain state
  \mbox{\hyperlink{namespacetransfer}{transfer}}                    Transfer AAC from \mbox{\hyperlink{structaccount}{account}} to 
      \mbox{\hyperlink{structaccount}{account}}
  net                         Interact with local p2p network connections
  \mbox{\hyperlink{namespacewallet}{wallet}}                      Interact with local \mbox{\hyperlink{namespacewallet}{wallet}}
  sign                        Sign a transaction
  push                        Push arbitrary transactions to the blockchain
\end{DoxyCode}
