\subsection{Detailed Description}
Contains comparison operators and stream operators for any P\+OD types that do not have their own operators. If P\+OD is comparable or streamable using it\textquotesingle{}s own operator or it\textquotesingle{}s conversion operator, then the original operator is used.

Just write {\bfseries using} {\bfseries namespace} {\bfseries flat\+\_\+ops}; and operators will be available in scope.

{\bfseries Example\+:} 
\begin{DoxyCode}
\textcolor{preprocessor}{#include <\mbox{\hyperlink{flat_2ops_8hpp}{boost/pfr/flat/ops.hpp}}>}
\textcolor{keyword}{struct }\mbox{\hyperlink{structcomparable__struct}{comparable\_struct}} \{      \textcolor{comment}{// No operators defined for that structure}
    \textcolor{keywordtype}{int} i; \textcolor{keywordtype}{short} s; \textcolor{keywordtype}{char} data[7]; \textcolor{keywordtype}{bool} bl; \textcolor{keywordtype}{int} a,b,c,d,e,f;
\};
\textcolor{comment}{// ...}

\textcolor{keyword}{using namespace }flat\_ops;

\mbox{\hyperlink{structcomparable__struct}{comparable\_struct}} s1 \{0, 1, \textcolor{stringliteral}{"Hello"}, \textcolor{keyword}{false}, 6,7,8,9,10,11\};
\mbox{\hyperlink{structcomparable__struct}{comparable\_struct}} s2 \{0, 1, \textcolor{stringliteral}{"Hello"}, \textcolor{keyword}{false}, 6,7,8,9,10,11111\};
assert(s1 < s2);
std::cout << s1 << std::endl; \textcolor{comment}{// Outputs: \{0, 1, H, e, l, l, o, , , 0, 6, 7, 8, 9, 10, 11\}}
