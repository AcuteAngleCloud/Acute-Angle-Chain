Detailed Description

template<typename MultiIndexType>
class chainbase::generic_index< MultiIndexType >

The value_type stored in the multiindex container must have a integer field with the name 'id'. This will be the primary key and it will be assigned and managed by generic_index.

Additionally, the constructor for value_type must take an allocator
