Detailed Description

template<typename T, typename U, typename Ret, typename... Inputs, typename ... Translated>
struct aacio::chain::webassembly::wavm::intrinsic_invoker_impl< Ret, std::tuple< array_ptr< T >, array_ptr< U >, size_t, Inputs... >, std::tuple< Translated... > >

Specialization for transcribing a pair of array_ptr types in the native method signature that share size This type transcribes into 3 wasm parameters: 2 pointers and byte length and checks the validity of those memory ranges before dispatching to the native method

Template Parameters
Ret- the return type of the native method
Inputs- the remaining native parameters to transcribe
Translated- the list of transcribed wasm parameters
