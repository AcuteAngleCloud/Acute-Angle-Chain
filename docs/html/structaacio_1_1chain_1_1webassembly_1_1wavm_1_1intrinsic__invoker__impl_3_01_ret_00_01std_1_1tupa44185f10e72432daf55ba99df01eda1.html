template<typename T, typename Ret, typename... Inputs, typename ... Translated>
struct aacio::chain::webassembly::wavm::intrinsic_invoker_impl< Ret, std::tuple< array_ptr< T >, size_t, Inputs... >, std::tuple< Translated... > >

Specialization for transcribing a array_ptr type in the native method signature This type transcribes into 2 wasm parameters: a pointer and byte length and checks the validity of that memory range before dispatching to the native method

Template Parameters
Ret- the return type of the native method
Inputs- the remaining native parameters to transcribe
Translated- the list of transcribed wasm parameters
